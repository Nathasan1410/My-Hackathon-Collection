## 🚀 Quick Start

### Adding a New Project

1. Create a folder under `projects/` with your project name
2. Copy the template from `projects/_template/README.md`
3. Fill in your project details
4. Update the main table above

### Project README Template

Each project README should include:
- **Overview**: What the project does
- **Tech Stack**: Languages, frameworks, tools used
- **Achievements**: Prizes, notable mentions
- **Links**: GitHub repo, live demo, Devpost submission
- **Learnings**: Key takeaways from the hackathon
