## Overview

**AlterEgo** is an AI-powered LinkedIn post generator that helps professionals create viral, engaging content tailored to their personal writing style. Built for the Encode Club's "Commit To Change" hackathon with $30,000 in prizes.

### Key Features

- **Style Analysis**: Analyzes user's writing from LinkedIn posts to learn their voice
- **Content Generation**: Generates topics, hooks, body, and CTAs using Groq LLaMA 3.3
- **Research Integration**: Web search via Tavily API for context and trending topics
- **Voice Input**: Audio transcription using OpenAI Whisper for hands-free input
- **Viral Scoring**: AI-powered content quality assessment
- **Multi-language**: Support for Indonesian and English
